# Portfolio-Optimization-Algorithmic-Trading-using-Deep-Reinforcement-Learning-FinRL-
Deep reinforcement learning trading pipeline (FinRL + Stable-Baselines3) that trains A2C, DDPG, PPO, TD3, and SAC agents on DJIA equities, benchmarks them against Mean-Variance Optimization and market indices, and achieves up to 18% excess returns vs. S&amp;P 500/Dow Jones
